The one-click weekly report

"A weekly report that writes itself."

Client: Northstar Preschools Stack: Node.js · Excel · Word (docx) Result: ~3 hours → minutes

Problem Every Friday, the same three hours

A weekly marketing report meant pulling numbers from the dashboard, formatting tables, writing up takeaways, and making it presentable. Every single week. The report mattered, leadership ran their Monday meeting on it, but producing it was pure manual assembly.

Build Data in, finished document out

  • A programmatic report generator. A Node.js build script reads the week's dashboard workbook and produces a polished, branded Word document: KPI cards, per-location performance tables, 8-week trends, and a takeaways section.
  • Consistent structure, week over week. Same sections, same formatting, same colour-coded statuses. Readers know exactly where to look, and week-to-week comparison is effortless.
  • Numbers stay honest. Every figure comes straight from the source workbook. No retyping, no transposition errors, no "which version is this?"
The result

Report production went from around 3 hours to minutes. Leadership gets it on time, every week, in the same clean format.
