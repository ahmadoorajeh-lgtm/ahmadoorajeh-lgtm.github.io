The KPI dashboard

"From raw export to answers in one tab."

Client: multi-location ad account Stack: Google Sheets · QUERY · auto-updating raw feed Result: answers in seconds

Problem The data existed. The answers didn't.

Raw ad platform data was landing in a spreadsheet, thousands of rows of it, but there was no usable view of performance. "How's campaign X doing this month?" meant filtering, pivoting, and double-checking by hand. Every question was a small data-pull project.

Build One tab that answers questions

  • QUERY-driven KPI tiles. Spend, leads, cost-per-lead and trends, computed live from the raw data tab. No manual refresh.
  • Breakdowns that answer real questions. Performance by campaign and by ad set, with date-range controls, so "how did this do last month?" is a dropdown, not a project.
  • A raw tab that updates itself. Fresh platform data flows in automatically; the dashboard recalculates the moment it lands.
The result

Performance questions answered in seconds instead of a data pull. The dashboard became the first tab open every morning.
